{\bfseries{Messaging Structure}}\begin{adjustwidth}{1em}{0em}
~\newline



\begin{DoxyItemize}
\item The messaging structure works as follows\+:
\begin{DoxyEnumerate}
\item Messager thread starts and reads from file
\item Messager thread sends messages to neurons and then waits {\ttfamily WAIT\+\_\+\+TIME} {\ttfamily WAIT\+\_\+\+INCREMENT} times
\item Each {\ttfamily Neuron\+Group} thread loops through its neurons and checks their activation status
\begin{DoxyItemize}
\item if activated the neuron runs, exhausts its message queue, and sends messages to any neighbor neurons (by adding to each respective queue)
\end{DoxyItemize}
\item After running through all its neurons, the {\ttfamily Neuron\+Group} waits {\ttfamily WAIT\+\_\+\+TIME} {\ttfamily WAIT\+\_\+\+INCREMENT} times.
\end{DoxyEnumerate}
\item The message queue is implemented as {\ttfamily std\+::list} because random access is not needed and popping from the front is neccessary.
